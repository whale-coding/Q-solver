## ✨ 核心功能

### 🛡️ 隐身模式 — "幽灵窗口"

| 特性 | 说明 |
|:---:|:---|
| 🚫 **录屏隐形** | 大部分截图/录屏软件无法捕获本窗口 |
| 👆 **鼠标穿透** | 可开启鼠标穿透，操作窗口后方的应用 |
| 📌 **始终置顶** | 可设置悬浮在其他窗口之上 |
| 🎯 **不抢焦点** | 尽量避免打断你正在进行的工作 |

> ⚠️ 以上功能在不同系统/软件环境下效果可能不同，**请务必在实际使用前自行测试确认**。

---

### 🎙️ Gemini Live API — 实时语音交互

> 💡 **使用场景**：实时捕获对方语音，AI 同步生成回答建议

| 特性 | 说明 |
|:---:|:---|
| 🗣️ **语音转录** | 实时捕获系统音频并转为文字 |
| 🤖 **即时回答** | AI 根据转录内容即时生成回答建议 |
| 🗺️ **问题导图** | 自动将对话整理成可视化思维导图 |
| 📤 **笔记导出** | 一键导出 Markdown 格式的对话笔记 |

---

### 🔌 多模型支持 — 灵活选择你的 AI

| 提供商 | 模型示例 | 特色 |
|:---:|:---|:---|
| **OpenAI** | GPT-4o, o1-preview | 通用能力强 |
| **Gemini** | gemini-2.0-flash-exp | 支持 Live API 实时语音 |
| **Claude** | Claude 3.5 Sonnet | 长文本理解出色 |
| **自定义** | 任意 OpenAI 兼容 API | 支持 OneAPI 等聚合服务 |

---

### 📄 上下文增强 — 个性化回答

- **📑 简历导入**：支持 PDF / Markdown 格式，AI 结合你的背景给出针对性回答
- **🧠 多轮记忆**：保持对话上下文，追问不用重复背景信息
- **✨ 智能解析**：一键将 PDF 简历解析为结构化 Markdown

## 🚀 快速开始

### 方式一：下载安装包（推荐）

<a href="https://github.com/jym66/Q-Solver/releases">
  <img src="https://img.shields.io/badge/下载-最新版本-blue?style=for-the-badge&logo=github" alt="Download">
</a>

> **macOS 用户注意**：下载后可能需要添加执行权限：
> ```bash
> chmod +x Q-Solver.app/Contents/MacOS/Q-Solver
> ```

### 方式二：源码编译

```bash
# 前置要求：Go 1.25+, Node.js 22+, Wails CLI
go install github.com/wailsapp/wails/v2/cmd/wails@latest

# 克隆并运行
git clone https://github.com/jym66/Q-Solver.git
cd Q-Solver
wails dev

# 编译生产版本
wails build -ldflags "-s -w" -tags prod
```

## ⚙️ 配置指南

1. 点击右上角 **设置** 图标
2. 选择 **提供商**：OpenAI / Gemini / Claude / 自定义
3. 填入 **API Key**
4. 选择 **模型**
5. （可选）导入 **简历** 获得个性化回答

### 支持的提供商

| 提供商 | 模型 | Live API |
|--------|------|----------|
| OpenAI | GPT-4o, o1 等 | ❌ |
| Gemini | gemini-2.0-flash-exp | ✅ |
| Claude | Claude 3.5+ | ❌ |
| 自定义 | 任意 OpenAI 兼容 API | ❌ |

## 🍎 macOS 设置

<details>
<summary><b>📸 截图权限</b></summary>

1. 打开 **设置** → **截图** 选项卡
2. 点击 **授权截图权限**
3. 在系统设置中勾选本应用
4. 返回点击 **刷新权限状态**

</details>

<details>
<summary><b>🎙️ 系统音频采集（Live API 需要）</b></summary>

macOS 需要安装虚拟音频驱动才能捕获系统音频：

```bash
# 安装 BlackHole
brew install blackhole-2ch
```

然后在 **音频 MIDI 设置** 中配置：
1. 创建 **多输出设备**
2. 添加 **BlackHole 2ch** + 你的扬声器
3. 设为系统输出
4. BlackHole 输入音量调至 **1.0**

<img src="assets/img5.png" width="500"/>

</details>
